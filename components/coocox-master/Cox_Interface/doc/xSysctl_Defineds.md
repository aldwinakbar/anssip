xACMP Peripheral Base Tab             {#xSystcl_ACMP_md}
====================
- 这组宏定义是ACMP设备的时钟开启状态, 是非强制类型，N表示该芯片不包含ACMP模块，不支持该功能。
- 使用了这个宏的API
    -  \ref xSysCtlPeripheralEnable <br>
    -  \ref xSysCtlPeripheralDisable <br>
